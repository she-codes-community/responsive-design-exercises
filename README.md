Responsive Design Exercises

A collection of small exercises used in the She Codes Responsive Design lesson.

Each folder focuses on a different responsive CSS concept and includes starter files for the exercise and a solution folder with one possible implementation.


Exercises

percent-layout  
Working with percentage units to create flexible layouts.

rem-units  
Using rem units for scalable typography and spacing.

viewport-units  
Exploring vw and vh to create elements that adapt to the viewport size.

media-query-layout  
Using media queries to adapt layouts for different screen sizes.

final-exercise-responsive-page  
A larger exercise that combines multiple responsive techniques.


These exercises are intended to be explored during the lesson and practiced in the browser using developer tools.
